データ型編集におけるUIの表記仕様
==========================

## 表記仕様

引数、戻り値、変数のデータ型は以下の表記を使用すること。

- **V**: void
- **I**: 整数 (Integer)
- **R**: 浮動小数点数 (Real number)
- **S**: 文字列 (String)
- **B**: 真偽値 (Boolean)
- **P**: プリプロセッサ (Preprocessor)
- **I[]**: 整数の配列
- **R[]**: 浮動小数点数の配列
- **S[]**: 文字列の配列
- **B[]**: 真偽値の配列
- **ui_\***: 任意のUIデータ型
- **\***: 任意のデータ型

## データ型の選択

複数選択が可能なUIコンポーネントを使用して、ユーザーがデータ型を選択できるようする。

- チェックボックス
- テキストによる直接指定

## 複数のデータ型を指定する場合

区切り文字 `||` を使用して複数のデータ型を表現する。

 例えば:

- `I||S||V`: 整数、文字列、void のいずれか
- `I[]||S[]`: 整数の配列または文字列の配列
