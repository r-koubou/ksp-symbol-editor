# 2026-03-27 作業ログ

## 作業内容

### Add / Bulk import 時の視覚的フィードバック

Add・Bulk import でシンボルを追加した際に、視覚的にわかりやすくする機能を全エディタ（variables・commands・callbacks・ui-types）に実装した。

#### 修正ファイル

**CSS（背景色の変更・newly-added クラス追加）**

- `apps/variables/style.css`
- `apps/commands/style.css`
- `apps/callbacks/style.css`
- `apps/ui-types/style.css`

各 CSS に以下の変更を適用：
- ホバー色を `#eff6ff`/`#bfdbfe` → `#e6f0fd`/`#a8c9f8` に変更
- アクティブ色を `#dbeafe`/`#93c5fd` → `#cde0fa`/`#78b2f5` に変更（視認性向上のため）
- `.newly-added` クラスを新規追加（背景 `#f0fdf4`、ボーダー `#86efac` の緑系）
- `.newly-added:hover`・`.newly-added.active` も定義

**JS（状態管理・スクロール）**

- `apps/variables/app.js`
- `apps/commands/app.js`
- `apps/callbacks/app.js`
- `apps/ui-types/app.js`

各 JS に以下の変更を適用：
- `newlyAddedIds = new Set()` をステートに追加
- `handleNewSymbol`: 作成した ID を `newlyAddedIds` に追加、`selectSymbol()` 後に `scrollIntoView` でスクロール
- `handleBulkImportConfirm`: 各シンボル作成後に ID を `newlyAddedIds` に追加、最後に追加したシンボルへスクロール
- `handleFileLoad`: ロード完了後に `newlyAddedIds.clear()` でリセット
- `renderSymbolList`: `newlyAddedIds` に含まれる ID の `li` に `newly-added` クラスを付与
