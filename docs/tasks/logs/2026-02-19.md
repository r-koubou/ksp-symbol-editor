#### 実装詳細

コマンドシンボルエディタをベースに、コールバック仕様（`callback-schema.yaml`）に合わせた差分実装。

**コマンドエディタからの主な変更点：**

| 変更箇所 | 内容 |
|----------|------|
| FormatVersion | `1.0.0`（クォートなし） |
| フォーム | `AllowMultipleDeclaration` チェックボックスを追加 |
| Return Type | form-card ごと削除 |
| 引数テーブル列 | Data Type → Required（RequiredDeclare の true/false 表示） |
| 引数モーダル | DataType + Quick Selection を削除し、RequiredDeclare チェックボックスに置き換え |
| エクスポートファイル名 | `callbacks.yaml` |

**YAML出力仕様：**
- `AllowMultipleDeclaration: true/false`（boolean、クォートなし）
- Arguments 内: `RequiredDeclare: true/false`
- 引数の `Description` は共通ルール準拠（空: `""`、非空: `|-` ブロック）

#### ユーザーによる修正

- `apps/callbacks/app.js`・`apps/commands/app.js`・`apps/variables/app.js` の `FormatVersion` 出力をクォートなし（`1.0.0`）に統一
- `apps/commands/app.js` の parseYaml デフォルト値を `'2.0'` → `'1.0.0'` に修正（仕様適合）
- `docs/specs/yaml-data-schema/variable-schema.yaml` の FormatVersion default を `"2.0"` → `"1.0.0"` に修正（仕様ミス訂正）
