### コールバックシンボルエディタの作成

`apps/callbacks/` ディレクトリにコールバックシンボルエディタを新規作成。

#### 成果物

- `apps/callbacks/index.html`
- `apps/callbacks/style.css`
- `apps/callbacks/app.js`

#### 実装詳細

コマンドシンボルエディタをベースに、コールバック仕様（`callback-schema.yaml`）に合わせた差分実装。

**コマンドエディタからの主な変更点：**

| 変更箇所 | 内容 |
|----------|------|
| FormatVersion | `1.0.0`（クォートなし） |
| フォーム | `AllowMultipleDeclaration` チェックボックスを追加 |
| Return Type | form-card ごと削除 |
| 引数テーブル列 | Data Type → Required（RequiredDeclare の true/false 表示） |
| 引数モーダル | DataType + Quick Selection を削除し、RequiredDeclare チェックボックスに置き換え |
| エクスポートファイル名 | `callbacks.yaml` |

**YAML出力仕様：**
- `AllowMultipleDeclaration: true/false`（boolean、クォートなし）
- Arguments 内: `RequiredDeclare: true/false`
- 引数の `Description` は共通ルール準拠（空: `""`、非空: `|-` ブロック）

#### ユーザーによる修正

- `apps/callbacks/app.js`・`apps/commands/app.js`・`apps/variables/app.js` の `FormatVersion` 出力をクォートなし（`1.0.0`）に統一
- `apps/commands/app.js` の parseYaml デフォルト値を `'2.0'` → `'1.0.0'` に修正（仕様適合）
- `docs/specs/yaml-data-schema/variable-schema.yaml` の FormatVersion default を `"2.0"` → `"1.0.0"` に修正（仕様ミス訂正）

## 動作確認結果

### UIタイプシンボルエディタの作成

`apps/ui-type/` ディレクトリにUIタイプシンボルエディタを新規作成。

#### 成果物

- `apps/ui-type/index.html`
- `apps/ui-type/style.css`
- `apps/ui-type/app.js`

#### 実装詳細

コールバックシンボルエディタをベースに、UIタイプ仕様（`ui-type-schema.yaml`）に合わせた差分実装。

**コールバックエディタからの主な変更点：**

| 変更箇所 | 内容 |
|----------|------|
| フォーム | `AllowMultipleDeclaration` を削除 |
| フォーム | `VariableType` プルダウン（V/I/R/S/B/P/I[]/R[]/S[]/B[] の10種）を追加 |
| フォーム | `RequireInitializer` チェックボックスを追加 |
| 引数配列キー | `Arguments` → `InitializerArguments` |
| 引数テーブル列 | Required 列を削除（Name / Description / Action の3列）|
| 引数モーダル | `RequiredDeclare` チェックボックスを削除（Name・Description のみ）|
| エクスポートファイル名 | `ui-types.yaml` |

**YAML出力仕様：**
- `VariableType: I`（文字列スカラー、必要時のみクォート）
- `RequireInitializer: true/false`（boolean、クォートなし）
- `InitializerArguments: []` または序列形式
- 引数: `Name` + `Description`（共通ルール準拠）

#### ユーザーによる修正

- `apps/ui-type/` ディレクトリを `apps/ui-types/` にリネーム
- `apps/ui-types/app.js` のエクスポートファイル名を `ui-types.yaml` → `uitypes.yaml` に修正

正常動作を確認。
